# ADHD 최적화 학습 루틴 가이드

## 핵심 원칙

### 1. 도파민 관리
- **즉각적 보상**: 작은 성취마다 XP와 피드백
- **시각적 진행**: 스트릭, 레벨, 뱃지로 동기 부여
- **다양성**: 같은 유형 작업 3회 이상 연속 금지

### 2. 에너지 기반 스케줄링
- **고에너지 시간**: 새 지식 학습, 어려운 개념
- **중에너지 시간**: 복습, 연습
- **저에너지 시간**: 가벼운 복습, 회고

### 3. 하이퍼포커스 관리
- **최대 45분** 연속 작업 후 강제 휴식
- **알람 필수**: 시각적 + 청각적 알림
- **작업 전환 알림**: 5분 전 경고

## 일일 루틴

### 아침 루틴 (5-10분)
```
1. 시스템 시작
   $ python src/main.py
   > start

2. 대시보드 확인
   - 오늘 복습 카드 수
   - 스트릭 상태
   - 일일 퀘스트

3. 에너지 레벨 자가 평가
   - 오늘 에너지: high/medium/low
```

### 학습 세션 (포모도로 기반)

#### 고에너지 세션 (오전)
```
[25분 작업] 새 지식 학습
   - 어려운 개념 습득
   - 정교화 질문 답변
   - 예시 만들기

[5분 휴식] 완전한 휴식
   - 화면에서 떠나기
   - 스트레칭
   - 물 마시기

[반복 4회 후 15분 긴 휴식]
```

#### 중에너지 세션 (오후)
```
[25분 작업] 복습 + 연습
   - 복습 카드 처리
   - 실습/적용
   - 연결 만들기

[5분 휴식]
```

#### 저에너지 세션 (저녁)
```
[20분 작업] 가벼운 복습
   - 쉬운 카드만
   - 회고/정리
   - 다음날 계획

[5분 휴식]
```

### 저녁 마무리 (5분)
```
1. 오늘 학습 기록 확인
   > report

2. 내일 학습 시간 확인
   > schedule

3. 스트릭 유지 확인
```

## 주간 루틴

### 월요일: 계획의 날
```
- 주간 학습 목표 설정
- 이번 주 집중 주제 선택
- 가용 시간 확인 및 스케줄링
```

### 화~금: 학습의 날
```
- 일일 루틴 수행
- 새 지식 추가 (하루 최대 20개)
- 복습 카드 처리
```

### 토요일: 실습의 날
```
- 배운 내용 실제 적용
- 프로젝트에 적용
- 예시 만들기
```

### 일요일: 회고의 날
```
- 주간 리포트 확인
- 어려웠던 개념 재학습
- 다음 주 계획
```

## ADHD 특화 전략

### 1. 시작 저항 극복
```
문제: 시작하기 어려움
해결:
- 2분 규칙: "2분만 해보자"
- 환경 준비: 학습 전용 공간
- 루틴화: 같은 시간, 같은 장소
- 바디더블링: 온라인 학습 그룹 활용
```

### 2. 집중력 유지
```
문제: 쉽게 산만해짐
해결:
- 방해 요소 제거: 폰 다른 방에
- 백색 소음/집중 음악
- 짧은 세션: 25분 → 15분도 OK
- 시각적 타이머 사용
```

### 3. 하이퍼포커스 관리
```
문제: 과몰입으로 다른 일 놓침
해결:
- 45분 하드 스톱
- 다중 알람 설정
- 스마트워치 진동 알림
- 작업 전환 5분 전 경고
```

### 4. 감정 조절
```
문제: 실수에 과민 반응
해결:
- 틀려도 XP 획득
- 실패 = 학습 기회
- 진행률 > 완벽함
- 어제의 나와 비교
```

## 자동화 설정

### macOS/Linux cron 설정
```bash
# 아침 9시 학습 리마인더
0 9 * * * cd /path/to/adhd-learning-system && python -c "from src.scheduler.automation import NotificationManager; NotificationManager().send_learning_reminder()"

# 저녁 8시 스트릭 경고
0 20 * * * cd /path/to/adhd-learning-system && python -c "from src.scheduler.automation import NotificationManager; NotificationManager().send_streak_warning(7)"
```

### Python 스케줄러 (항상 실행)
```python
from src.scheduler.automation import AutomationRunner

automation = AutomationRunner()
automation.setup({
    "daily": {
        "morning_reminder": "09:00",
        "evening_warning": "20:00"
    }
})
automation.start()
```

## 문제 해결

### "오늘 학습할 기분이 아니야"
→ 2분만 해보기. 정말 싫으면 그만둬도 됨.
→ 최소 1장만 복습해도 스트릭 유지!

### "복습 카드가 너무 많아"
→ 에너지 높은 시간에 집중
→ 하루 최대 한도 설정 (기본 60분)
→ 새 카드 줄이기

### "새 지식이 안 들어와"
→ 청킹: 더 작게 쪼개기
→ 기존 지식과 연결하기
→ 예시 많이 만들기

### "스트릭이 끊겼어"
→ 괜찮아! 복귀 뱃지 획득 가능
→ 오늘부터 다시 시작
→ 1일차도 100일차도 같은 하루
